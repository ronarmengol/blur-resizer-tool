You can upload most common image formats, like JPG, PNG, GIF, and WEBP. You can choose to export your processed images as JPG, PNG, or WEBP using the "Output format" dropdown on the tool page.

ü§î What do the different output images mean?

  • Compressed Original: This is your original image, run through a compression algorithm. It keeps the same dimensions but should have a smaller file size.
  • Blurred Desktop: This is your original image, also compressed, with a 15px blur filter applied.
  • Mobile (640px): Your image is resized so its width is 640 pixels, and it is compressed.
  • Mobile Blurred (640px): Your image is resized to 640px wide, a 15px blur is applied, and it is compressed.
